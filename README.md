# Valo-Feedback Mirror

YouTube 動画を見ながら、Valorant の反省メモをタイムスタンプ付きで残せる完全無料の Web アプリです。
データはすべてブラウザの LocalStorage に保存されるため、ログイン不要・バックエンド不要で動作します。

## 🚀 起動方法

以下のコマンドを実行することで、ローカルサーバーが起動します。

```bash
npm install
npm run dev
```

起動後、ブラウザで [http://localhost:3000](http://localhost:3000) にアクセスしてください。

## ✨ 機能紹介

### 1. YouTube プレイヤー統合
- 上部の入力欄に YouTube の URL を貼り付けるだけで動画が読み込まれます。
- YouTube IFrame API を使用した精密な操作が可能です。

### 2. 反省ログ機能
- メモ入力欄に文字を入力して **Enter** を押すと、現在の再生時間を取得してログに追加します。
- ログをクリックすると、動画がその時間へ即座にシークします。
- **「コピー」ボタン**: Discord 等に貼り付けやすい形式で全ログをクリップボードにコピーできます。
- **「インポート」ボタン**: 外部で整形したログテキスト（例: `[01:23] ここが惜しい`）を貼り付けて一括取り込みが可能です。`[hh:mm:ss]` やブラケットなしの形式にも対応しています。

### 3. ワイプ拡大機能（右下ズーム）
- **9等分ズーム**: 画面を3x3の9等分に分けた際の、右下1ブロック分を画面いっぱいに拡大表示します（scale 3x）。
- **コントロール干渉回避**: YouTubeの標準ボタン（全画面等）を邪魔しない位置に、Searchアイコン付きのフローティング起動ボタンを配置しています。
- **直感操作**: ズームエリアをクリックで拡大、拡大中は画面のどこをクリックしても元に戻ります。

### 4. キーボードショートカット
- **Space**: 再生 / 一時停止
- **← (左矢印)**: 5秒戻る
- **→ (右矢印)**: 5秒進む
- **Enter**: メモ入力（入力欄にフォーカスがある場合のみ）

### 5. 安心の設計
- **自動保存**: LocalStorage を使用しているため、ページをリロードしてもログが消えません。
- **誤操作防止**: ログが1行以上ある場合、ページを閉じたりリロードしようとするとブラウザの離脱確認が表示されます。

## 🛠️ 技術スタック
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Video API**: YouTube IFrame Player API
